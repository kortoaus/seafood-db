# 수산물 지식 베이스 구축 요약

## 이게 뭔가요?

수산코너 운영에 필요한 **어종 정보**를 체계적으로 모은 데이터베이스입니다.

어떤 생선을 매입했을 때, 그 생선이 뭔지(이름, 특성), 어떻게 보관하고 손질하는지, 경쟁 업체들은 어떤 형태로 팔고 있는지를 한눈에 볼 수 있도록 정리했습니다.

**총 200종**의 어종 정보가 등록되어 있고, 현재 리포트 작성이 진행 중입니다.

## 어떻게 가공했나?

### 1단계: 어종 마스터 목록 작성

위의 모든 자료에서 어종명을 추출하고, 중복을 제거하여 **200종**의 마스터 목록을 만들었습니다.

각 어종에는 다음 정보가 포함됩니다:
- 영문명 (Australian standard name)
- 한국명 (FishBase 학명 매칭 — 50종만 확인됨, 나머지는 미확인)
- 학명 (Scientific name)
- 과(Family) 분류
- 검색 키워드 (영문 + 한글, 유사 표현 포함)
- 관련종 연결

### 2단계: 한국명 매칭

- FishBase (국제 어류 데이터베이스)에서 **학명 기반**으로 한국 표준명을 매칭
- **학명이 정확히 일치하는 경우만** 한국명으로 등록 (50종)
- 나머지 150종은 한국 공식 표준명이 없거나, 같은 속(genus)의 다른 종이라 정확하지 않아 등록하지 않음
- AI가 추측한 이름은 사용하지 않음 — **잘못된 정보보다는 없는 게 낫습니다**

### 3단계: 어종별 리포트 생성 (진행 중)

수집한 모든 자료를 종합하여, 어종별로 다음 내용이 포함된 리포트를 자동 생성합니다:

- **기본 정보**: 영문명, 한국명, 학명, 검색 키워드
- **특성**: 생김새, 구별법
- **크기 및 중량**: 유통 규격
- **보관법**: 냉장/냉동 보관 기간
- **손질 및 조리**: 손질 방법, 추천 조리법, 궁합 식재료
- **상품화 형태**: 경쟁사들이 실제로 이 생선을 어떤 형태(냉장/냉동/훈제/가공)로, 어떤 포션(200g, 1kg 등)으로, 어떤 소비 형태(회, 구이, 찜 등)로 팔고 있는지
- **대체 가능 어종**: 이 생선이 없을 때 뭘로 대체할 수 있는지
- **관련종 비교**: 비슷한 종과의 차이점

### 출처 원칙

**모든 정보에는 반드시 출처가 표기됩니다.** 출처가 없는 정보는 싣지 않고, 자료가 없으면 "자료 없음"으로 표기합니다.

## 제약사항 (반드시 읽어주세요)

**이 리포트는 AI(Claude)가 자동 생성한 문서입니다.**

수집한 원본 자료(SFM, DPI, 상품 데이터 등)는 사실이지만, 그것을 종합하여 리포트로 작성하는 과정에서 AI가 **잘못 해석하거나 부정확하게 정리했을 가능성이 있습니다.**

구체적으로:
- **한국명 매칭 오류** — 학명이 정확히 일치하는 50종 외에는 한국명이 틀릴 수 있음
- **관련종/대체종 분류 오류** — 같은 과(Family)라고 해서 실제로 대체 가능한 것은 아님
- **상품화 형태 분류 오류** — 상품명에서 어종을 추출할 때 다른 어종의 상품이 섞였을 수 있음
- **보관 기간/조리법** — SFM 원본 데이터를 번역한 것이나 번역 과정에서 뉘앙스가 달라졌을 수 있음
- **전반적인 정확도** — 검증된 리포트가 아니므로, 중요한 판단(가격 책정, 식품 안전 등)에는 반드시 별도 확인이 필요합니다

**이 자료는 참고용이며, 최종 판단은 현장 경험과 전문가 확인을 거쳐야 합니다.**

## 앞으로 할 일

1. **AI 상품화 도우미 연동** — "킹피시 15kg 매입했어" → AI가 리포트 기반으로 상품화 방안 제안
2. **수율 데이터 보강** — 부위별 수율(필렛 몇%, 머리 몇%)은 공개 자료가 거의 없어 현장 경험 기반으로 축적 필요
3. **리포트 지속 검증** — 현장에서 사용하면서 잘못된 정보 발견 시 수정
